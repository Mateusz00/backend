Accounts

registerNewAccount

Creates a new account, use returned JWT from Authorization header for subsequent requests

Request Body schema: application/json
username
required
string (AccountUsername)
password
required
string (AccountPassword)

It must be at least 8 characters long and contain 1 character from each of the following categories: uppercase letter, lowercase letter, number and special character.

email
required
string (AccountEmail)

Responses

localhost:8080/accounts

  • Shell + Curl
